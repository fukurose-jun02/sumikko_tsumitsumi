# 🐾 すみっコつみつみ！

UFOキャッチャー風クレーンがすみっコをお皿に落としてくる、かわいい積み上げゲームです。

## 🎮 遊び方

- クレーンが右上のおもちゃ箱からすみっコを取り出してランダムな位置に落とします
- **左右キー** または **マウス移動** でお皿を動かしてキャッチ！
- スマホは画面をスライドまたは下の ⬅️ ➡️ ボタンで操作
- お皿の上にそーっと積み上げて、**30秒間**でなんびき乗せられるか挑戦！
- 急にお皿を動かすとキャラがふわっと落ちてしまうよ😨

## ✨ ゲームの特徴

- 本物のすみっコぬいぐるみ画像を使用
- UFOキャッチャー風クレーンアニメーション
- ふわふわ物理演算でキャラが積み重なる
- お皿に縁（へり）があるので少し安定
- 4歳のお子さんでも遊べる優しい難易度
- 時間が経つほどクレーンの動きが速くなる

## 🚀 GitHub Pages で公開する方法

1. このリポジトリを GitHub に作成
2. ファイルをすべてアップロード
3. Settings → Pages → Branch を `main` に設定 → Save
4. `https://ユーザー名.github.io/リポジトリ名/` で公開完了！

## 🛠️ カスタマイズ

`game.js` の上部にある定数を変えるだけで調整できます：

```js
const PLATE_W = 150;      // お皿の幅
const CHAR_R  = 26;       // キャラのサイズ
const GRAVITY = 0.16;     // 重力（小さいほどゆっくり）
```
